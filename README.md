# Marubatu Game (3×3)

HTML / CSS / JavaScript で作成した三目並べゲームです。  
スマートフォンでの操作を前提に、シンプルなUIで実装しています。

## 遊び方
- マスをタップして ◯ / × を置きます
- 「はじめから」をタップするとリセットされます

## 特徴
- ボタン要素を使わないUI設計
- スマホ操作前提
- JavaScriptのみでロジック実装

## 注意
本作品はスマートフォン向け設計のため、  
PCブラウザではクリック可能な領域が分かりにくい場合があります。

## 技術要素
- DOM操作
- イベント処理
- 勝敗判定ロジック
