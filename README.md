# マルバツゲーム（初期作）

これは、2023年に制作した JavaScript のマルバツゲームです。  
HTML + CSS + JS で構成されており、スマホ向けに作成しています。

## アルゴリズムの流れ（簡略版）
1. ゲーム開始時に先攻/後攻を設定
2. クリックされたマスにマークを反映
3. CPUはリーチ判定を行いつつ、乱数で手を選択
4. 勝敗判定を行い、ゲーム終了時に表示
5. リセットボタンで全マス初期化

## 目的・背景
- JavaScript の学習目的も兼ねた初期創作物
- 2000年代前半に趣味サイトをやっていたのが JavaScript に触れた始まりで、そこから長いブランクがあります
- 2021年にプログラミング学習を再開し、最初は Android Studio を使っていたものの2022年に中断
- アルゴリズムの基礎を学ぶために JavaScript に立ち返り、ウェブ上で遊べるゲームとして作成

## 技術要素
- DOM操作
- イベント処理
- 勝敗判定ロジック

## 注意点（初期作のため）
- UI はスマホ向けで、PC で見るとクリック可能領域や操作性が分かりにくい所があります
- コードはベタ書きが多く、コメントも大量に入っています
- 先攻・後攻表示やクリック領域など、改善の余地があります
- 公開にあたり一部修正を行いました(function startGameUIを新しく作りコードを簡潔にした等)

## 遊び方
- 表示されるマスをクリックして「○」や「×」を置きます
- 先攻・後攻はゲーム開始時に切り替わります
- 「ゲームを中断してはじめから」ボタンでリセットできます
